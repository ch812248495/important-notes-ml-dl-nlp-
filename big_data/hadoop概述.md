# hadoop 概述

## 什么是 hadoop
- 开源的
- 可靠的
- 可扩展的
- 分布式计算

## 解决的问题
- 海量数据的存储 
    - HDFS
    - 存储海量数据
    - 动态添加机器
    - 备份三份, 当数据文件损坏, 快速恢复
- 海量数据的分析
    - mapreduce
    - 核心理念:分而治之
- 集群资源的管理和任务调度
    - yarn
    - 资源管理( cpu, 内存)
    - 任务调度
## 特点
- 扩容能力
- 成本低
- 高效率
- 可靠性

## 背景
三大论文
- GFS - HDFS
- mapreduce
- bigtable - HBASE

## 应用场景
- 日志分析
- 推荐系统
- GPS 数据分析推送
